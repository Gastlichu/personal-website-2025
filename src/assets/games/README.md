# Game Preview Images

Place game preview/screenshot images here.

## Expected files:
- `rocket-upgrade.png` (or .jpg) — Screenshot for Rocket Upgrade
- `seance-sphere.png` (or .jpg) — Screenshot for Séance Sphere

## Recommended dimensions:
- **Aspect ratio:** 16:9 (e.g., 1280×720 or 1920×1080)
- **Format:** PNG or JPG
- **Size:** Keep under 500KB for fast loading

## How to enable:
Once you add an image, uncomment the `image:` line in the corresponding game's markdown file:

```markdown
# Before:
# image: ../../assets/games/rocket-upgrade.png

# After:
image: ../../assets/games/rocket-upgrade.png
```
